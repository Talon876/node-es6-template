# Project

This is a template for ES6 node modules.

## QuickStart

Create a new a folder then

```sh
curl -fsSL https://github.com/Talon876/node-es6-template/archive/master.tar.gz | tar -xz --strip-components=1 node-es6-template-master
npm install
git init
```
